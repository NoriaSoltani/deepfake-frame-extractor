# Deepfake Frame Extractor

This project extracts frames from videos, resizes them, and saves them in
a structured folder format. It can process any folder containing videos
and automatically creates subfolders for each file.

## Features
- Automatic detection of project paths
- Extract frames from all videos in a folder
- Resize frames to any resolution (default: 224Ã—224)
- Clean and simple project structure
- Works on Windows, Linux, and macOS
